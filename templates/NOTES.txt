Your n8n local stack has been deployed.

1. Add these hosts to /etc/hosts (point at your ingress controller):

     127.0.0.1 {{ .Values.n8n.publicUrl.host }} {{ .Values.ingress.grafanaHost }}

{{- if .Values.ingress.tls.enabled }}

2. Create the local TLS secret with mkcert if you have not already:

     brew install mkcert
     mkcert -install
     mkcert {{ .Values.n8n.publicUrl.host }} {{ .Values.ingress.grafanaHost }}

     kubectl -n {{ .Values.namespace }} create secret tls {{ .Values.ingress.tls.secretName }} \
       --cert={{ .Values.n8n.publicUrl.host }}+1.pem \
       --key={{ .Values.n8n.publicUrl.host }}+1-key.pem

{{- end }}

3. Open n8n:

     {{ .Values.n8n.publicUrl.editorBaseUrl }}

   Open Grafana:

     {{ .Values.n8n.publicUrl.protocol }}://{{ .Values.ingress.grafanaHost }}/

4. Check pod status:

     kubectl get pods -n {{ .Values.namespace }}

5. View n8n logs:

     kubectl logs -n {{ .Values.namespace }} deploy/n8n-main --tail=100
